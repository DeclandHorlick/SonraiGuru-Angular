About Sonrai-Guru

A banking web application which displays users transaction data on informative charts allowing users to gauge their spending and select a savings goal. Combining angular JS and HTML with MuleSoft to communicate easily with the BBVA API, and then manipulate this data to display in our graphs. The project aims to give users more detail on their spending patterns and allow them to create achievable savings goals based on their account information. We hope that this application could also be used by corporations to gauge their expenditure and then use this information to increase money efficiency. The graphs display in multiple formats and change based on the period of time you would like to view. The User can only access their graphs 'dashboard' when they are logged in and can logout from the dashboard page.
